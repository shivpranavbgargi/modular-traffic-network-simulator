# Modular Road Traffic Simulator

A small modular **road traffic generator and simulator** for directed road networks. It models junctions, roads, road-side output queues, traffic sources, destinations, adaptive routing, congestion, source throttling, and animated visual output.

The simulator is intended to show how cars move through a road graph when each junction has limited outgoing queue space. Cars are generated by sources, routed through junctions, wait in output queues when exits are busy, travel along roads with finite capacity, and eventually leave through destination sink junctions.

---

## TLDR: how the traffic works

Think of the simulation as a directed road map:

```text
source -> junction -> output queue -> road -> junction -> output queue -> road -> destination
```

At every tick:

1. **Sources try to generate cars.**
   A source creates cars using either a constant interval or a Poisson arrival process.

2. **A new car is only accepted if there is a usable path to its destination.**
   If the source junction has no available outgoing route because all useful output queues are full, the source does not inject the car for that tick. This is source throttling.

3. **Cars choose an outgoing road at each junction.**
   The simulator computes a route to the destination and chooses the best next road. If the preferred output queue is full, it tries another valid outgoing route.

4. **Cars wait in output buffers at junctions.**
   Each junction has one queue per outgoing road. These are road-side queues, like cars waiting at an exit lane.

5. **Roads move cars forward.**
   A road has finite capacity and travel time. A car can enter a road only if the road has space.

6. **Cars exit when they reach their destination junction.**
   Once a car reaches its destination sink, it is counted as completed.

## What the dots around nodes mean

The dots around a junction are **cars waiting in that junction's output buffers**.

They are not random decoration and they are not cars currently travelling on the road.

| Visual element                          | Meaning                                                                                                   |
| --------------------------------------- | --------------------------------------------------------------------------------------------------------- |
| Moving colored circles on roads         | Cars currently travelling on roads                                                                        |
| Small colored dots near a junction      | Cars waiting in that junction's output queue                                                              |
| Dots near a specific outgoing direction | Cars waiting to take that output road                                                                     |
| `+N` near an output path                | More than 10 cars are waiting on that output side, so the simulator collapses the extra cars into a count |
| Road color green/yellow/red             | Road occupancy / congestion level                                                                         |

Example:

```text
        +7
        o
        o
        ↑
      [J1] ─────► road
```

This means cars are queued at junction `J1` for the outgoing road in that direction. The `+7` means there are 7 additional cars beyond the visible dots.

## Main features

* Directed road network with named junctions and roads
* Finite road capacity and per-road travel time
* Per-junction **output buffers** instead of unbounded input buffers
* Configurable output buffer capacity
* Optional per-junction output buffer capacity from JSON
* Source throttling when no usable route currently exists
* Adaptive routing based on congestion
* Automatic rerouting when a preferred output queue is full
* Anti-U-turn behavior to reduce immediate oscillation
* Multiple traffic sources and multiple destinations
* Constant and Poisson traffic generation modes
* Animated GIF output
* Per-run JSON statistics
* PDF statistics report with queue, throughput, wait-time, and road-occupancy plots

## Visualization guide

### Moving circles

Cars travelling on roads

### Dots near nodes

Cars waiting in output queues

### `+N`

Collapsed count for large queues

### Road colors

Green → light traffic
Yellow → moderate
Red → heavy congestion

## Summary

This is a **road traffic generator**, not a network simulator.

It models:

* cars moving through a road map
* queues forming at exits
* congestion affecting routing
* sources reacting to available capacity

All behavior emerges from:

* road capacity
* junction output buffers
* adaptive routing
* source throttling
